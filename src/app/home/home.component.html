Welcome

Welcome to FieldID!

This app was created to solve a real world problem by managing user tracking in a venue

Adding new users

You can add new users by going to Manage Users. From there you can type in his details and upon registration a unique QR code will be created. The user uses this QR code everytime he enters the venue

Scanner

You, as the venue receptionist can scan the QR codes of the users that are entering the venue by simply using the scanner

Record

Record keeps track of the users you have scanned sorting them by most to less recent
